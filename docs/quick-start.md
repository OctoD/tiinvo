# Quick start

## Installing tiinvo

To install tiinvo 

```bash
yarn add tiinvo
```

or if you use npm

```bash
npm i tiinvo
```

if you are using deno just import it

```ts
import { } from 'https://cdn.skypack.dev/tiinvo'
```
